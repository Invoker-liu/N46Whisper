# 变更：集成开源 TTS 框架实现文本转语音功能

## 为什么
当前项目仅支持视频字幕提取（语音转文字），用户希望扩展反向功能——将提取的字幕或其他文本内容转换为语音文件，形成完整的语音-文字双向转换能力。

## 变更内容
- 集成开源 TTS 框架到 Google Colab 环境
- 复用现有文件选择器组件和交互逻辑
- 支持多种语音类型选择（男声/女声、不同语言、不同音色）
- 支持从本地上传或 Google Drive 选择文本文件
- 输出常见音频格式（.mp3、.wav）
- 利用 Colab GPU 加速 TTS 推理
- 实现长文本分段处理和进度显示

## 影响
- 受影响规范：新增 `tts` 功能模块
- 受影响代码：`N46Whisper.ipynb`（新增单元格）、可能新增 `tts_utils.py`
- 依赖变更：新增 TTS 框架依赖（待技术选型确定）
